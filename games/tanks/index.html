Diesel Tanks

So this is a really basic tanks game. The idea is that playing games with friends is fun. So we put a tanks game on the screen. wire it up to the internet and then people can control their tank with their cell phone

At this point you shoudl open up a copy of the server page. If you have a chromecast you should chrome cast it. then all the players open the client page and enter the game code and should just show up.

If somehthing goes wrong drop me a note.
