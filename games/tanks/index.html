Diesel Tanks

So this is a really basic tanks game. The idea is that playing games with friends is fun. So we put a tanks game on the screen, wire it up to the internet and then people can control their tank with their cell phone

At this point you should open up a copy of the server page. If you have a chromecast you should chrome cast it. Then all the players open the client page and enter the game code and should just show up.Players who join late will be added at the next round start.

If somehthing goes wrongor you want to give me feed back drop me a note.

Handy Tips

client.sounds.enabled =true in console on the server will enable sound
