Diesel Tanks

So this is a really basic tanks game. The idea is that playing games with friends is fun. So, we put a tanks game on the big screen, wire it up to the internet, and then people can control their tank with their cell phone.

Good times.

How to get it going

At this point you should open up a copy of the server page on whatever device people can all see. If you have a chromecast you can chrome cast it. Then all the players open the client page and enter the game code and should just show up on the list.

Players who join late will be added at the beginning of the next round.

Requirements

  • Web Sockets This is the big one the game uses websockets to connect the clients and the server. You can check you have them here. Note that some versions of the browser in older versions of android lie about supporting web sockets. Chrome for Android does work if you have having mysterious issues.
  • HTML5 canvas The server page really needs this. The client does to but you can probably get away with out it with soemc areful quicnting and lcuk on knowing what weapons you have.

Handy Tips

  • diesel.game.sounds.enabled =true in console on the server will enable sound. These are awful. That's why they are disabled by default.
